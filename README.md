# 🌳 Decision Tree Framework

A professional web-based Decision Tree Framework developed using **Python**, **Flask**, **Scikit-learn** .The application allows users to upload datasets, build customizable Decision Tree models, evaluate performance, and visualize the generated tree through an intuitive web interface.

---

## 📌 Project Overview

This framework is designed to simplify Decision Tree model development by providing an easy-to-use interface for data upload, preprocessing, visualization, and performance evaluation.

The application supports multiple datasets and enables users to configure important Decision Tree parameters such as maximum depth and maximum leaf nodes without writing any code.

---

## ✨ Features

- 📂 Upload CSV datasets
- 🔍 Automatic variable identification
  - Numerical variables
  - Continuous variables
  - Categorical variables
  - outlier treatment of variables
  - one hot encoding if necesaary
- 🎯 Target variable selection
- 🌲 Decision Tree generation
- ⚙️ User-defined Maximum Depth
- 🍃 User-defined Maximum Leaf Nodes
- 📊 Model Accuracy
- 🎯 Precision Score
- 📈 Confusion Matrix
- 🌳 Decision Tree Visualization
- 📋 Rule Generation
- 📑 Dataset Summary
- 📉 Feature Analysis
- 🎨 Interactive and user-friendly web interface

## 📋 Workflow

1. Upload a CSV dataset.
2. Review the dataset summary.
3. Select the target variable.
4. Select predictor variables.
5. Configure:
   - Maximum Depth
   - Maximum Leaf Nodes
6. View:
   - Accuracy
   - Precision
   - Confusion Matrix
   - Decision Tree
   - Generated Rules
